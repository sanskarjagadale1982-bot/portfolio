# 🌐 Personal Portfolio Website

This is my personal portfolio website where I showcase my projects, skills, blogs, and contact information.

The goal of this website is to create a professional online presence and share my work with others.

---

## 🚀 Features

- Responsive design (works on mobile and desktop)
- About Me section
- Skills section
- Projects showcase
- Blog section
- Contact form
- Smooth animations with JavaScript

## 🛠️ Technologies Used

Frontend:

- CSS3
- JavaScript

Backend:
- Python (for contact form handling)

Other Tools:
- Git

- GitHub Pages
